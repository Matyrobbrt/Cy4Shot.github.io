MCMeta Animations

1. Introduction

Did you ever want to have an animated texture in Minecraft? Well, you can using .mcmeta files, which is a format that Mojang has made specifically for animating Minecraft textures.

2. Creating a .mcmeta file

We can start by creating a file called texture_name.png.mcmeta in the same folder as our texture (if the texture is called bin_item.png the mcmeta will be called bin_item.png.mcmeta).

Now we can add this into the .mcmeta file:


{
    "animation": {
        "interpolate": true,
        "width": 16,
        "height": 16,
        "frametime": 10,
        "frames": [
            0,
            1,
            2, // And so on... here you loop your frames.
        ]
    }
}

Lets break that code down:

  • interpolate: (Optional) If true, Minecraft generates additional frames between frames with a frame time greater than 1 between them. Defaults to false.
  • width: (Optional) The width of the tile, in pixels. This is unused in vanilla's files but can be used by resource packs to have frames that are not perfect squares. Defaults to 16
  • heigth: (Optional) The height of the tile, in pixels. This is unused in vanilla's files but can be used by resource packs to have frames that are not perfect squares. Defaults to 16
  • frametime: Sets the default time for each frame in increments of one game tick.
  • frames: Contains a list of frames. Defaults to displaying all the frames from top to bottom. A number corresponding to position of a frame from the top, with the top frame being 0.
