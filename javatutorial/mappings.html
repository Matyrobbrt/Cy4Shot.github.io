Mappings

Mappings are basically what defines a functions name. By default, in 1.16.5, forge uses Mojmaps, but my tutorials use MCP. MCP have less functions mapped, but they usually have more user friendly descriptions.

1. Changing mappings

If you would like to change, simple do the following steps: + Go to your mod's folder and open CMD. + Type gradlew -PUPDATE_MAPPINGS=20201028-1.16.3 -PUPDATE_MAPPINGS_CHANNEL=snapshot updateMappings and wait for the process to finish. + Locate your build.gradle file in your project. + Open the file and find <mappings channel: 'official', version: '1.16.5'>. + Replace it with <mappings channel: 'snapshot', version: '20201028-1.16.3'>. + Save the file. + Right click on your project and go to gradle>refresh. + Wait for the refresh to complete.

2. Using Mojmaps

Alternatively, you can use the bot in #bot-commands in our discord. Simply type !mcpmm <name_of_function> 1.16 in order to convert from the MCP used in my tutorials to the Mojmaps used by 1.16.5.
