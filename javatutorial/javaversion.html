Wrong Java Version

When starting the game, you may see that it tries to start the game with Java 15/16, even though Java 8 is installed. To fix this, we need to edit some project settings.

1. Locating installed JREs

Right click on your project, and click on Properties. Next, go down to Java Build Path, and in the properties window, select the Libraries tab. Click JRE System Library and then click Edit on the left. Finally, click Installed JREs.

2. Changing build path

Now, click Add, select Standard VM and hit Next. Next, click Directory and locate the installation path of the Java 8 JRE. If you have downloaded OpenJDK as I have done in my video tutorials, you will find the path under C:\Program Files\AdoptOpenJDK\jdk-8.0.282.8-hotspot\jre. Once you have done that, click Finish. Then, tick the box thats next to the Java 8 JRE, and click Apply and Close. Click Alternate JRE, and select the Java 8 JDK from the dropdown. Finally, click Finish and then Apply and Close.

3. Refresh gradle

To rebuild gradle, right click on your project and go to Gradle > Refresh Gradle Project, and wait for the progress bar in the bottom right to finish.
